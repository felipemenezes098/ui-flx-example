# Web – Flx example

Next.js app for this example. The main repo documentation is in the [root README](/README.md).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Set up `.env.local` from `.env.example` (see the root README).

## Scripts

- `npm run dev` – development server
- `npm run build` – production build
- `npm run typegen` – generate TypeScript types from the Sanity schema
